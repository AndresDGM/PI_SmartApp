Conceptos Algebra Lineal

Operaciones básicas con matrices

Suma de matrices


La suma de dos matrices A y B del mismo tamaño m x n se define como la matriz C
de tamaño m x n cuyos elementos son la suma de los elementos correspondientes
de A y B. Es decir, para cada elemento cij de C, se tiene que cij = aij + bij
para i = 1, 2, ..., m y j = 1, 2, ..., n.


Multiplicación por un escalar


La multiplicación de una matriz A por un escalar k se define como la matriz B de
tamaño m x n cuyos elementos son el producto de k por los elementos
correspondientes de A. Es decir, para cada elemento bij de B,
se tiene que bij = k * aij para i = 1, 2, ..., m y j = 1, 2, ..., n.


Producto de matrices


El producto de dos matrices A y B es posible si el número de columnas de A es
igual al número de filas de B. El producto AB es una matriz C de tamaño m x p
cuyos elementos se calculan utilizando el método del producto punto entre filas
y columnas: cij = a1j * b1i + a2j * b2i + ... + anj * bni para i = 1, 2, ...,
m y j = 1, 2, ..., p.


Transposición de matrices


La transposición de una matriz A de tamaño m x n es la matriz At de tamaño n x m
cuyos elementos son los mismos que los de A pero intercambiados los índices i y j.
Es decir, aijt = aji para i = 1, 2, ..., n y j = 1, 2, ..., m.
