Conceptos Algebra Lineal

Vectores


Un vector es una entidad matemática que tiene magnitud (longitud o tamaño) y
dirección. Se puede representar como una flecha en el espacio, donde la longitud
de la flecha representa la magnitud y la dirección indica la dirección del vector.
Los vectores se utilizan para representar cantidades físicas como desplazamiento,
velocidad, fuerza, entre otros.


Magnitud del vector


La magnitud de un vector se calcula utilizando la fórmula de la norma o longitud
del vector. En dos dimensiones, la magnitud de un vector (x, y) se calcula
utilizando la fórmula:

|v| = √(x^2 + y^2)
En tres dimensiones, la fórmula es similar, pero se agrega la componente z:
|v| = √(x^2 + y^2 + z^2)

Dirección del vector


La dirección de un vector se puede determinar utilizando las componentes del vector.
En dos dimensiones, la dirección se puede obtener utilizando la función trigonométrica
arcotangente (tan^(-1)):
θ = tan^(-1)(y / x)
Donde θ es el ángulo entre el vector y el eje x positivo.

Sentido del vector


El sentido del vector se determina observando las componentes del vector. Si las
componentes son positivas, el vector apunta en la dirección positiva de cada eje
correspondiente. Si las componentes son negativas, el vector apunta en la dirección
negativa de cada eje correspondiente.

Producto punto (producto escalar)


El producto punto es una operación entre dos vectores que resulta en un número
escalar. Se calcula multiplicando las componentes correspondientes de los vectores
y sumándolas. La fórmula del producto punto para dos vectores en tres dimensiones
(x, y, z) es: A · B = Ax * Bx + Ay * By + Az * Bz. El producto punto se utiliza
para calcular la proyección de un vector sobre otro, encontrar ángulos entre vectores
y determinar si dos vectores son ortogonales (su producto punto es cero).

Producto cruz (producto vectorial)


El producto cruz es una operación entre dos vectores que resulta en un tercer vector
perpendicular a los dos vectores originales. La fórmula del producto cruz para dos
vectores en tres dimensiones (x, y, z) es:
A x B = (Ay * Bz - Az * By, Az * Bx - Ax * Bz, Ax * By - Ay * Bx).
El producto cruz se utiliza para calcular áreas de paralelogramos y volúmenes de
paralelepípedos, determinar la dirección de una normal a un plano y resolver sistemas
de ecuaciones lineales.

Espacios vectoriales


Un espacio vectorial es un conjunto de vectores que cumple todas las propiedades de un
subespacio vectorial. Además de las propiedades mencionadas anteriormente, un espacio
vectorial debe contener un conjunto de vectores linealmente independientes que generen
todos los demás vectores del espacio. Esto significa que cualquier vector del espacio
se puede expresar como una combinación lineal de los vectores generadores. Los espacios
vectoriales son fundamentales en álgebra lineal y se utilizan para estudiar propiedades
geométricas y algebraicas de los vectores y sus operaciones.

Subespacios vectoriales


Un subespacio vectorial es un conjunto de vectores que cumple ciertas propiedades.
Debe contener el vector cero, ser cerrado bajo la suma de vectores y la multiplicación
por un escalar. Esto significa que si tomas dos vectores del subespacio y los sumas,
el resultado también debe estar en el subespacio. Además, si tomas un vector del subespacio
y lo multiplicas por un escalar, el resultado debe estar en el subespacio. Los subespacios
vectoriales pueden ser planos, rectas o incluso el espacio completo.
