Como moverse?

El movimiento del jugador de realiza mediante las teclas wasd y shitf
-la tecla "w" para moverse arriba
-la tecla "a" para moverse a la izquierda
-la tecla "s" para moverse abajo
-la tecla "d" para moverse a la derecha
-la tecla shift para moversemas rapido mientras se precione

Como atacar?

Para atacar debes apuntar con el cursor a tu objetivo y usar el click izquierdo unas vez hecho esto tu personaje disparara un proyectil en esa direccion

Como parar el juego?

con la tecla ESC o con el boton de la barra lateral puedes pausar el juego esto desplegara un menu para poder reiniciar, reanudar o salir del juego

cual es el objetivo?

Tu objetivo es derrotar a todos los enemigos en el mapa una vez lo hagas te aparecera el menu de pausa pero con el mensaje de "Ganaste!!!" y te dara la opcion de volver a jugar o salir del juego
